## 1. Product page description

A Max for Live performance surface for Novation Launch Control XL MK3
· 15 custom modes — instant recall, no menu diving
· Cross-mode jumps: momentary peek or permanent switch
· Mute, Solo, mixer, instruments — one tap each
· Ready-to-play Live Set with built-in walkthrough
· Max for Live device — drop into any Live project

────────────────────────────
**WHAT IT DOES**

Play your whole rig like a single instrument. M4L device for Novation Launch Control XL MK3. Modes 1–10 control your instruments. Modes 11–14 become a dedicated mixer, plus a cue/prelisten mode. One hotkey jumps between both sides, returning you to the exact page you left.

Two encoder banks per channel extend each channel to 6 controls — without breaking muscle memory. Hold a button to preview the next layer. Turn the encoder to stay there. Switch modes without reaching for the controller.

────────────────────────────
**WHO IT'S FOR**

Live performer — peek into another layer, or switch permanently when needed. Momentary and toggle switching make transitions fast and predictable.
Studio producer — move between writing, sound design and mixing with a single button, without breaking the session flow.
New LCXL owner — open a ready-to-play Live Set with preconfigured mappings and an interactive walkthrough.

────────────────────────────
**WHAT'S IN THE BUNDLE**

• Control XL — the Max for Live device (the update check is built in)
• 15 preconfigured Custom Modes for the LCXL MK3 (10 instruments, 4 mixer, 1 cue) — .syx, import via Novation Components
• Ableton project — Demo (with sounds) or Starter (empty); pick your download
• Quickstart guide (PDF)
• Free v1.x updates via Gumroad

────────────────────────────
**REQUIREMENTS**

• Ableton Live 11 or 12 — Suite, or Standard with the M4L add-on
• Max for Live 8.5 or newer
• Novation Launch Control XL MK3
• Novation Components (browser app) for mode loading
• macOS or Windows — wherever Live 11 or 12 runs

Not compatible with LCXL MK1 or MK2 — they use a different mode-switching architecture.

────────────────────────────
**LICENSING**

One license key. Up to 3 activations across your machines.

────────────────────────────
**UPDATES & SUPPORT**

Gumroad emails a download link whenever a new version ships. No subscription. Bug reports and questions: support@fadercraft.com — response within 48 hours on working days.

────────────────────────────
**REFUNDS**

Refunds available within 14 days of purchase if the device does not work as described. Contact support@fadercraft.com with your license key and a brief description of the issue. See https://fadercraft.com/refund for full policy.

## 2. Post-purchase receipt ("Plain Text" content note)

Welcome to Fadercraft Control XL.

Two things before you start — install, then wire MIDI. The Quickstart below covers both.

**QUICKSTART**

1. Drop Control XL.amxd onto an empty MIDI track in Ableton Live.
2. On that track set MIDI From → All Ins · All Channels, MIDI To → Launch Control XL MK3 (DAW) · Ch. 7, Monitor → In. Without this the device receives no input from the controller — the single most common "it doesn't work" cause.
3. Move any fader or turn a knob on the LCXL. The device lights up and responds. You're done.

**WHAT'S INSIDE**

— Control XL — the Max for Live device. The update check is built in — no extra files to place.
— custom-modes/ — 15 Launch Control XL MK3 Custom Modes (.syx). Import via Novation Components (one file per mode).
— the Ableton project (.als) — device, mappings, and (Demo) a demo set.
— Quickstart.pdf — this guide, printable.

**UPDATES**

The device checks for new versions on launch. When an update ships, a "New Version" link appears inside the device — your license key (in your receipt above) unlocks the download. All future v1.x updates are free.

**SUPPORT**

Email support@fadercraft.com — replies within 48 hours.
Refunds within 14 days, no questions asked.
