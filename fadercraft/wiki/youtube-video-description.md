## Description (current draft)

Fadercraft Control XL is a performance system for Ableton Live designed for fast, tactile interaction with your session.

- 14 channels with 6 controls each
- Page-based control system for workflow separation
- Global hotkeys for instant switching
- Performance-oriented sends and FX routing
- Designed for live improvisation and fast interaction

Fadercraft is designed to remove friction between intention and control — keeping you focused on performance instead of navigation.
