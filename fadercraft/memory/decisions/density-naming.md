# Decision — Density migration (закрыто 2026-05-15)

Статус: **migration выполнена**

Универсальное правило именования density-токенов живёт в директиве `ux-ui-designer` (раздел «Density-шкала: правило именования»). Этот файл фиксирует, как Fadercraft пришёл к нему.

## Что было

- В `tokens.css` использовались имена `--d-*` со слитной записью дроби: `--d-025` (= 1px), `--d-125` (= 5px), `--d-125dense` (= 50px — с суффиксом `dense` для обхода коллизии `1.25 ↔ 12.5`).
- Параллельно жил слой `--space-0..10` с дублирующими значениями (raw px).
- 11 Text Styles в Figma, 19 typography-стилей в коде — разрыв 8 стилей.
- Три значения, используемых в typography (17, 23, 25 px), отсутствовали в density.

## Что сделано

1. **Зафиксировано правило именования** в директиве `ux-ui-designer`. Имя density-токена — только число, без префиксов и суффиксов. Точка как разделитель дроби в Figma, дефис в CSS. Параллельный `space`-слой избыточен — удалить.
2. **Добавлено 5 новых ступеней:** `4.25` (17px), `5.75` (23px), `6.25` (25px) — под Typography; `15` (60px), `20` (80px) — под секционные отступы.
3. **Префикс `--density-` снят** в CSS — переменные стали `--0`, `--0-25`, `--6`, `--12-5` и т.д.
4. **Удалён `--d-125dense`** и весь `--d-*` legacy-слой (был as transitional aliases, после миграции consumers — снят).
5. **Удалён `--space-*` слой целиком.** Numeric `--space-1..10` сначала переписан в t-shirt (`--space-2xs..5xl`), потом полностью убран — пользователь отверг отдельный spacing-слой как избыточный.
6. **Все consumers замигрированы** (≈80 файлов CSS + tokens.ts): теперь используют только `var(--N)` и семантические алиасы `var(--radius-*)`, `var(--page-pad)`, `var(--page-max)`.
7. **`tokens.ts`** переписан: блок `space` → `density` с числовыми ключами (`"0.25"`, `"6"`, `"12.5"`). Точка в TS-ключах допустима — это JS-нотация дробных чисел, не Figma-имя.
8. **Память агента** обновлена: `tokens/spacing.md` → `tokens/density.md`, карта памяти и горячие токены в `MEMORY.md` синхронизированы.

## Что осталось

**Закрыто 2026-05-15 (Этап 3):**
- `figma.config.json` обновлён: `collections.spacing → collections.density`, `"Spacing" → "Density"`.
- Запятая как разделитель дроби в Figma — зафиксирована как универсальное правило в директиве `ux-ui-designer` → «Жёсткие правила (универсальные)». Точка в variable names запрещена Figma Plugin API.
- 4 «extras» Text Styles в Figma (`Heading/Display`, `Heading/Subtitle`, `Body/Large`, `Body/Medium`) — решение **keep** (маркетинговый резерв, не в каноничном наборе из 15).
- **Разрыв Typography 11 ↔ 19 стилей** — нужно либо добавить 8 стилей в Figma, либо консолидировать code-side. Отдельная задача (см. `tokens/typography.md`).
