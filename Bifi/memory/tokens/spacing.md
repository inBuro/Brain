# Spacing

Источник: см. [tokens/density.md](density.md)
Статус: pointer (Beefy DS не имеет отдельной semantic spacing-коллекции — spacing задаётся напрямую через density)

## Архитектурная заметка

В Beefy DS **нет отдельной коллекции `Spacing`**. Spacing-значения берутся из примитивной шкалы Density (либо напрямую, либо через семантические алиасы из других коллекций — это ещё уточняется). То есть в дизайне `gap = 2` (= 8px из коллекции Density), а не `spacing.sm`.

Это сознательное решение: одна примитивная шкала вместо нескольких семантических слоёв. Меньше уровней косвенности, проще понять «откуда взялось число».

См. [tokens/density.md](density.md) для полной шкалы, naming, scopes и правил.

## Типичные применения density как spacing

| Случай | Token (Density) | px |
|---|---|---|
| Hairline / 1px gap | `0,25` | 1 |
| Минимальный gap между близкими элементами | `1` | 4 |
| Стандартный inline gap (icon + text) | `2` | 8 |
| Standard padding в input/button | `2` – `3` | 8–12 |
| Card padding | `4` | 16 |
| Section vertical spacing | `6` – `8` | 24–32 |
| Большие группы / hero | `10` – `12` | 40–48 |

(Это рекомендация, а не правило — сверяй с реальными компонентами в Beefy.)
