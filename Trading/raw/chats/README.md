# raw/chats/

Дистилляты разговоров с Claude как источник для wiki.

## Формат файла

Имя файла: `YYYY-MM-DD-короткое-название.md` (lowercase, через дефис).

Содержимое каждого файла:

```markdown
# Заголовок дистиллята

**Дата разговора**: YYYY-MM-DD
**Участники**: куратор + Claude (модель)
**Тема**: одна-две строки про предмет разговора
**Статус**: draft | ingested

---

## Контекст

Зачем был разговор, какую задачу решали.

## Ключевые тезисы

- Тезис 1
- Тезис 2

## Сырой материал

Цитаты, фрагменты переписки или итоговые выводы — то, что должно остаться immutable как источник.
```

## Правила

- Файлы в этой папке **immutable**. Если нужны правки — создаётся новый файл с новой датой.
- После ingest в wiki статус меняется на `ingested` (это единственная допустимая правка — пометка статуса).
- Каждый файл должен породить хотя бы одну страницу-саммари в `wiki/`.
