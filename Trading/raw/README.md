# raw/

Папка с неприкосновенными исходными документами.

## Правила

- **Никогда не модифицировать** содержимое этой папки. Источники должны быть immutable, чтобы цитаты в wiki оставались верифицируемыми.
- Если источник устарел или заменён — добавляется новый файл, старый остаётся.
- Каждый файл в `raw/` должен иметь хотя бы одну страницу-саммари в `wiki/`.

## Подпапки

- `raw/chats/` — дистилляты разговоров с Claude как источник для wiki. Формат описан в `raw/chats/README.md`.

Дополнительные подпапки появляются по мере необходимости (например, `raw/papers/`, `raw/reports/`).

## Цитирование

См. секцию Citation rules в корневом `Claude.md`. Любой факт в wiki, опирающийся на источник из этой папки, должен сопровождаться пометкой `(source: filename.md)`.
