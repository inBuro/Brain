# accountant — память

Ремесло личного бухгалтера-аналитика. Только правила и методика, **никаких реальных цифр** (сумм, балансов, номеров счетов) — они живут вне `~/Brain`.

- [Парсинг выписок Bangkok Bank](bbl-statement-parsing.md) — venv+pdfplumber, колонка «Баланс» как позвоночник, сверка с контрольными итогами банка, regex строки операции
- [Карта категорий](categorization-rules.md) — метка операции Bangkok Bank → категория трат/дохода; правило аренды; почему «Мария»/конкретного получателя автоматически не выделить
- [Рецепт дашборда](dashboard-recipe.md) — один HTML + вендоренный Chart.js + data.js, тёмная тема, секции, интерактив, проверка через headless browse

## Главные принципы (продублированы из агента, чтобы не забыть)
1. Реальные финансовые данные — только вне `~/Brain` (по умолчанию `/private/tmp/spending/`). `~/Brain/.claude/` пушится в git.
2. Точность доказывается сверкой с итогами банка (0 расхождений), опора — колонка баланса.
3. В эту память — только правила, не цифры.
4. Чего нет в выписке (имя получателя перевода) — не выдумывать; спрашивать или давать интерактивную пометку.
