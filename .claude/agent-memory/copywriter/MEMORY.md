## Product

Fadercraft — umbrella-бренд утилит для живого выступления в Ableton Live (Max for Live). Флагман: **XL_Performance** — M4L-устройство для **Novation Launch Control XL MK3**. 16 custom-modes → два слоя (микшер 11–14 + инструменты 1–10), переключение одной кнопкой, Solo Follower, без MIDI-конфликтов. Цена $39. Storefront: Gumroad. Лендинг: fadercraft.com.

Planning / wiki: `~/Brain/fadercraft/`. Код лендинга: `~/Projects/Claude/Fadercraft/app/` (React).

## Voice (из brief.md — держать дословно)

Два атрибута по порядку: **технично/точно**, затем **прагматично/без булшита**.
- Цифры вместо прилагательных («Modes 11–14», не «several modes»).
- Механизм → выгода. Сначала физическое действие/роутинг, выгода следует.
- Никакого хайпа второго лица («you'll love this», «transform your sets» — запрещено).
- Одна мысль — одно предложение.
- Точные имена железа/софта: «Launch Control XL MK3», «Ableton Live Suite», «Max for Live», «Custom Modes», «Components».
- Называть исходную проблему прямо: какую конкретную трение снимает фича, а не какую мечту обслуживает.
- Changelog — это тоже копи: лабораторные заметки, честно, без полировки.
- Правило проверки: прочитай вслух. Звучит как пресс-релиз запуска → переписать как строку из docs.

## Audience / ICP

- **Primary:** live-перформер на Ableton (клубы, фестивали, стримы). Владеет LCXL MK3 ради тактильного контроля. Боль: переключение частей сета — один жест, не охота за курсором. Ценит: одна кнопка без разрыва flow, предсказуемый фидбэк (LED, fader pickup, без MIDI-конфликтов), надёжность на 90-мин сете.
- **Secondary:** студийный продюсер + хардвар-контроль; M4L-энтузиаст с maxforlive.com.

## Market signal (см. [[wiki/audience-reddit]])

- **Firmware V1.1 (29 янв 2026)** добавил fader pickup, кривые энкодеров, выбор Custom Mode удержанием Mode. → **Главное возражение лендинга:** «зачем Fadercraft, если V1.1 это уже дал?». Ответ: firmware = сырые примитивы, Fadercraft = готовая сет-архитектура (слои + Solo Follower + без конфликтов). Показывать явно.
- Подтверждённые боли MK3: OLED «shows nothing», энкодеры «too stiff» для сцены, Custom Modes только через Components («not very friendly»).
- Язык аудитории (сырьё): «it shows nothing», «too stiff», «you have to go into the Components app», «not very friendly», «pick up its previous position rather than jumping».
- ⚠️ «Solo Follower», «два слоя», «один жест», «не разрывая flow» — наш бренд-нарратив, НЕ речь аудитории. Не выдавать за цитаты.

## Сильные месседжи (подтверждены болями)

1. «Custom Modes без Components» — сильнейшая зацепка.
2. «Предсказуемый фидбэк» — контрит боль OLED.
3. «Сделано под live, не под студию».
