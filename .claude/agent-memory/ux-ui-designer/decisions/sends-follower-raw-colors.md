# Sends Follower — raw hex цвет на eyebrow

**Решение (2026-06-24):** eyebrow-текст «TRACK» и «RETURN» в компоненте `Frame 5` (`2505:21696`) использует raw hex `#a1f7df` — НЕ трогать в рамках текущей задачи по адаптивности.

**Почему:** цвет не совпадает с каноническим `Primitives/Mint/400` (`#63f2ca`) — это чуть более светлый оттенок. Возможно намеренный выбор для контраста на тёмном фоне в этом конкретном контексте. Требует продуктового решения: либо зафиксировать как отдельный токен (`Primitives/Mint/200` или `color/text/brand-light`), либо выровнять с существующим минтом.

**Технический долг:** raw hex в Figma-ноде нарушает правило «все цвета — только через переменные». Требует:
1. Оценить визуальную разницу `#a1f7df` vs `#63f2ca` в контексте
2. Принять решение: новый токен или замена на существующий
3. Привязать к переменной

**Как применять:** до принятия решения — не заменять `#a1f7df` автоматически на `#63f2ca`. Разница визуально заметна.
