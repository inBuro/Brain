# Fadercraft pricing tiers (canonical)

Reference price points for revenue/conversion modeling and any analysis that touches device price. Set by the user 2026-06-25; default ladder for all products — deviate only with explicit reason.

| Tier | What it is | Price |
|---|---|---|
| Micro Utility | One action, one function, very specific narrow problem | $9 |
| Utility | Small complete tool (incl. a bundle of 2 related devices) | $14 |
| Workflow Tool | Larger tool that noticeably changes the workflow | $24 |
| Flagship | Big multi-feature system, high depth | $39 |
| Pro Bundle / Suite (if it ever exists) | Several large products in one package | $59–79 |

Examples: Sends Follower (Return + Track = 2 related devices) → Utility $14. XL_Performance → Flagship $39.
